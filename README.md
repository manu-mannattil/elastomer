## Usage

The Python scripts in this repository require the standard Python
scientific stack ([NumPy][numpy] + [SciPy][scipy]).  To build the
executables, you'll need a recent C++ compiler and the FFTW3 library.
Plotting requires [Matplotlib][mpl] and [charu][charu].
